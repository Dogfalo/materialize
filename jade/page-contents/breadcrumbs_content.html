Breadcrumbs are a good way to display your current location. This is usually used when you have multiple layers of content.
For accessibility purpose it has to be an ordered list with a dedicated label (optional).

Basic

You are here:

  1. First
  2. Second
  3. Third 



  <nav>
    <div class="nav-wrapper">
      <p id="breadcrumblabel" class="sr-only">You are here:</p>
      <ol class="breadcrumbs col s12" aria-labelledby="breadcrumblabel">
        <li class="breadcrumb">
          <a href="#!">First</a>
        </li>
        <li class="breadcrumb">
          <a href="#!">Second</a>
        </li>
        <li class="breadcrumb">
          <a href="#!">Third</a>
        </li>
      </ol>
    </div>
  </nav>
